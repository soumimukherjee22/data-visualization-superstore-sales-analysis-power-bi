# 📊 Superstore Sales Analysis – Power BI Dashboard

## 🧠 Project Overview
This Power BI dashboard analyzes **Superstore sales performance** to uncover insights on sales, profit, and customer behavior.  
It helps identify top-performing regions, product categories, and time periods, enabling data-driven decision-making.

## 🔍 Key Insights
- **West region** achieved the highest sales, while **Central region** showed areas for improvement.  
- **Office Supplies** generated the most sales volume.  
- **Technology category** provided the highest profit margins.  
- **November–December** were the top-performing months in sales.
